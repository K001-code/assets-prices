# assets-price-tracker

A lightweight, efficient tool to fetch live prices for precious metals and cryptocurrencies. This program leverages the Gold-API.com interface to provide up-to-the-minute market data.

The program tracks the following assets in real-time:
Precious Metals: Gold (XAU), Silver (XAG), Platinum (XPT), Palladium (XPD), and Copper (CPX).
Cryptocurrencies: Bitcoin (BTC), Ethereum (ETH).

Options for different types of measurement available depending on the asset, e.g. Gold (XAU).
