# assets-price-tracker

A lightweight, efficient tool to fetch live prices for precious metals and cryptocurrencies. This program leverages the _[gold-api](https://gold-api.com/)_ interface to provide up-to-the-minute market data.

## The program tracks the following assets in real-time:

🏅 Precious Metals: Gold (XAU), Silver (XAG), Platinum (XPT), Palladium (XPD), and Copper (HG).

🪙 Cryptocurrencies: Bitcoin (BTC), Ethereum (ETH).

Types of measurements: Ounce(oz), Gram(g), Kilogram(Kg), ជី(ji)
